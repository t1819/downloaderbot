# Download video/audio in your mobile phone 
DownloaderBot will configure your termux for downloading any video or audio in your phone.

# Preinstalled:
* Install Termux application from google play store: 
	- https://play.google.com/store/apps/details?id=com.termux
* Install python into Termux app.
	```python
	pkg install python3
	```
* Install Git package:
	```python
	pkg install git
	```
# How to Install?  

* Clone Github repo.
	```python
	git clone https://github.com/t1819/downloaderbot.git
	```
	
* Run setup file for configure your termux.
	```python
	python source/setup.py
	```

* Restart your termux & a new command will be available in your termux named as "downloaderbot".

# How to Use?

* Below are the example of command uses 
	```python
	# change the video_url text with original url.
	downloaderbot -t video -l video_url
	```

# Need commands Help?
* For search the available options
	```python
	downloaderbot -h
	```
*Enjoy downloading...*
